Comparison of Machine Learning Models for Diabetes Prediction

Project Overview :
This project compares multiple machine learning models to predict the likelihood of diabetes using medical diagnostic data. The goal is to evaluate the performance of different algorithms and identify the most effective model for diabetes prediction.

Dataset : 
Pima Indians Diabetes Dataset    
The dataset contains medical information such as glucose level, BMI, age, blood pressure, and other health indicators used to predict diabetes.

Technologies Used :
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow

Models Implemented :
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- AdaBoost
- Deep Neural Network (DNN)
- Ensemble Methods

Project Workflow :
1. Data loading and exploration  
2. Data preprocessing and cleaning  
3. Feature scaling and preparation  
4. Model training using different algorithms  
5. Model evaluation and performance comparison  
6. Analysis of prediction results  

Evaluation Metrics :
The models were evaluated using the following metrics:
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

Project Files :
"diabetes_project.ipynb" – main project notebook  
"diabetes.csv" – dataset used for training and testing  

Conclusion :
The project compares several machine learning models to understand their effectiveness in predicting diabetes. The analysis helps identify which models perform better for medical prediction tasks.

Future Improvements :
- Hyperparameter tuning for better model performance  
- Use of larger medical datasets
